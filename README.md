# 🎮 ARCADE B.ARENA

A simple and visually styled HTML5 Canvas arcade game built using **JavaScript, HTML, and CSS**.  
Control the paddle, bounce the ball, score points, and level up as the speed increases!

---

## 🚀 Features

- 🟢 Start Menu UI with neon effects  
- 🎵 Background music with toggle ON / OFF  
- 🔊 Sound effects (hit & game over)  
- 🎯 Score & High Score tracking (Local Storage)  
- 📈 Dynamic Level System (speed increases with score)  
- 🎨 Custom Ball Color (via options menu)  
- ⏸ Pause / Resume system (`P` key)  
- 🏆 High Score display screen  

---

## 🎮 Gameplay

- Use ⬅ Left Arrow and ➡ Right Arrow to move the paddle  
- Bounce the ball to earn points  
- Missing the ball = Game Over  
- Game gets faster as you level up  

---

## 🕹 Controls

| Key | Action |
|-----|--------|
| ⬅ / ➡ | Move Paddle |
| **P** | Pause / Resume |
| Mouse Click | Menu navigation |
